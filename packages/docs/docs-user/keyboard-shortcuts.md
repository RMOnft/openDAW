# Keyboard Shortcuts

Replace `⌘` with `Ctrl` on Windows and Linux.

## Misc

- `SPACE`: Toggle playback
- `⌘Z`: Undo
- `⌘⇧Z`: Redo
- `⌘S`: Save project

## Views

- `⇧0`: Show Dashboard
- `⇧1`: Show Classic View
- `⇧2`: Show Mixer
- `⇧3`: Show Modular Systems
- `⇧4`: Show Project Info

## Panels

- `B`: Toggle **B**rowser
- `E`: Toggle **E**ditor
- `D`: Toggle **D**evices
- `M`: Toggle **M**ixer

## Timeline

- Hold `⌘` and click to cut a region
- Hold `⌥` and drag to copy a region/clip
- Hold `⌥⇧` and drag to copy a region/clip as mirrored
- Click track header and hit `⌫` to delete track

## Editing

- `⌘C`: Copy
- `⌘V`: Paste
- `⌘D`: Duplicate selection
